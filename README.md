# MutantCreepers-No-Explody

A Fabric Minecraft mod that disables the explosive tendencies of Mutant Creepers.

## Overview

Mutant Creepers are terrifying, but sometimes you just want them to stick around without turning your base into a crater. This mod provides a surgically precise fix to prevent them from exploding while maintaining their other behaviors.

## Features

*   **Explosion Neutralization**: Prevents Mutant Creepers from triggering their signature blast.
*   **Vanilla Feel**: Keeps all other mutant mechanics intact.
*   **Lightweight**: Minimal performance impact, designed for compatibility.

## Installation

1.  Ensure you have **Fabric Loader** and **Fabric API** installed for Minecraft 1.20.1.
2.  Requires the original **Mutant Monsters** mod to be present.
3.  Drop this mod into your `mods` folder.

## Development Setup

This project uses Gradle. To set up the workspace:

1.  Clone the repository.
2.  Open in **VS Code** (recommended).
3.  Run `./gradlew genSources` to generate Minecraft source code.
4.  Run `./gradlew runClient` to test.
